# TF-IDF

Gebruik TF-IDF voor een content-based *News Recommender System*

TF-IDF is een manier om teksten om te zetten in vectoren. Dit maakt het mogelijk om de inhoud van krantenartikelen met elkaar te vergelijken.

[download notebook](/notebooks/tf-idf.ipynb)


### Start Jupyter

* Zorg ervoor dat je de juiste conda-omgeving hebt geactiveerd:

        conda activate RecSys

* Start Jupyter:
        
        jupyter lab

* Open vervolgens de notebook
